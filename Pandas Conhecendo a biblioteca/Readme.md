# 📊 Análise de Dados com Pandas — Fundamentos Aplicados

Este repositório demonstra, na prática, competências fundamentais de análise de dados em Python utilizando **Pandas**.  
Os materiais foram desenvolvidos a partir do curso **Introdução ao Pandas (Alura)** e evoluíram para experimentações próprias focadas em manipulação, limpeza e exploração de dados.

> 🎓 Curso concluído em: **18/02/2026**  
> ⏱️ Carga horária: **8 horas**


---

## 🛠️ Habilidades demonstradas

- Importação de múltiplos formatos
- Seleção eficiente de colunas e linhas
- Filtros condicionais
- Criação de variáveis derivadas
- Tratamento de valores nulos
- `groupby` multi-nível
- Métricas descritivas
- Tabelas resumo
